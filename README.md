# 🍗 McChicken - Panel Administrativo

Sistema de gestión administrativa con geolocalización desarrollado para la clase de Aplicaciones Web Orientadas a Servicios (AWOS).

## 🚀 Tecnologías

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Leaflet.js
- **Backend:** Java, REST API, Tomcat
- **Arquitectura:** MVC con separación por capas (Modelo, Core, REST, Frontend)

## ✨ Funcionalidades 
- Estoy trabajando para agregar más funcionalidades 😸
- 🗺️ **Mapa interactivo** con Leaflet.js (centrado en León, Gto)
- 📍 **Geolocalización de sucursales** con marcadores personalizados
- 👥 **CRUD de empleados** con cards dinámicas
- 🎨 **Diseño responsive** con paleta morada pastel minimalista
- 🔗 **Consumo de API REST** para datos dinámicos

## 📁 Estructura del Proyecto
McChicken/
├── McChicken_Modelo/     # Entidades y modelado de datos
├── McChicken_Core/       # Conexión a base de datos y controllers
├── McChicken_REST/       # Servicios REST/API
└── McChicken_Frontend/   # Interfaz de usuario


## 🛠️ Cómo ejecutar

1. Desplegar `McChicken_REST` en Tomcat
2. Abrir `McChicken_Frontend/index.html` en navegador
3. Asegurar que el servidor REST esté corriendo en `localhost:8080`
